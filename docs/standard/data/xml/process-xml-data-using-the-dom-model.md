# Elaborazione di dati XML con il modello DOM
Il DOM \(Document Object Model\) XML considera i dati XML come un set standard di oggetti e viene usato per elaborare i dati XML in memoria.  Lo spazio dei nomi `System.Xml` fornisce una rappresentazione programmatica di documenti, frammenti, nodi o set di nodi XML.  Si basa sulle raccomandazioni principali del World Wide Web Consortium \(W3C\) DOM Level 1 e DOM Level 2.  
  
 La classe <xref:System.Xml.XmlDocument> rappresenta un documento XML.  Include membri per il recupero e la creazione di tutti gli altri oggetti XML.  Usando la classe <xref:System.Xml.XmlDocument> e le classi correlate Ã¨ possibile costruire documenti XML, caricare e accedere ai dati, modificare i dati e salvare le modifiche.
