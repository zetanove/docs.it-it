[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Nelle tabelle seguenti vengono elencate tutte le parole chiave del linguaggio [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)].  
  
## Parole chiave riservate  
 Le parole chiave riportate di seguito sono *riservate*, ovvero non è possibile utilizzarle come nomi degli elementi di programmazione, ad esempio variabili o procedure.  Per ignorare questa restrizione, è possibile racchiudere il nome tra parentesi quadre \(`[` `]`\).  Per ulteriori informazioni, vedere "Nomi forzati" in [Declared Element Names](../../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
> [!NOTE]
>  Si consiglia di non utilizzare nomi forzati, poiché il codice potrebbe risultare di difficile lettura e potrebbero verificarsi errori gravi non facilmente rilevabili.

## Parole chiave non riservate  
 Le parole chiave riportate di seguito non sono riservate, ovvero è possibile utilizzarle come nomi per gli elementi di programmazione.  Si consiglia tuttavia di evitare questo tipo di utilizzo delle parole chiave non riservate, poiché il codice potrebbe risultare di difficile lettura e potrebbero verificarsi errori gravi non facilmente rilevabili.
