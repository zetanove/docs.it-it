# Previsto &#39;Next&#39;
Un costrutto `On Error Resume` si verifica senza un oggetto `Next` di conclusione.  
  
 **ID errore:** BC30003  
  
### Per correggere l'errore  
  
-   Aggiungere la parola chiave `Next` alla fine dell'istruzione `On Error`.
