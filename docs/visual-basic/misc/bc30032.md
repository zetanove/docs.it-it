# Gli eventi non possono avere un tipo restituito
Gli eventi possono accettare argomenti ma non possono restituire direttamente i valori.  
  
 **ID errore:** BC30032  
  
### Per correggere l'errore  
  
-   Rimuovere il tipo restituito dall'istruzione `Event`.
