# L&#39;etichetta &#39;&lt;nomeetichetta&gt;&#39; &#232; gi&#224; definita nel metodo o nell&#39;espressione lambda su pi&#249; righe corrente
Il nome dell'etichetta viene duplicato all'interno della routine. Tutti i nomi all'interno di una routine devono essere univoci.  
  
 **ID errore:** BC30094  
  
### Per correggere l'errore  
  
1.  Rinominare l'etichetta duplicata.
