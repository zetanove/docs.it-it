# Impossibile creare il file temporaneo nel percorso &#39;&lt;nomefile&gt;&#39;: &lt;messaggio di errore&gt;
Il compilatore [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] chiama Assembly Linker \(Al.exe, definito anche Alink\) per generare un assembly con un manifesto. Il linker ha rilevato un errore durante la creazione di un file o durante la scrittura di una risorsa in memoria in un file. Potrebbe trattarsi di un problema di configurazione.  
  
 **ID errore:** BC30138  
  
### Per correggere l'errore  
  
1.  Esaminare il messaggio di errore riportato e vedere l'argomento [Errori e avvisi di Al.exe](http://msdn.microsoft.com/it-it/7f125d49-0a03-47a6-9ba9-d61a679a7d4b) per spiegazioni e suggerimenti aggiuntivi.  
  
2.  Se l'errore persiste, raccogliere informazioni sulla situazione contingente e informare il Servizio Supporto Tecnico Clienti Microsoft.
