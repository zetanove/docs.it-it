# Non &#232; possibile creare l&#39;oggetto Assembly Linker: &lt;messaggioerrore&gt;
Il compilatore [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] chiama Assembly Linker \(Al.exe, definito anche Alink\) per generare un assembly con un manifesto. Il linker ha segnalato un errore di creazione del proprio oggetto di lavoro. Potrebbe trattarsi di un problema di generazione di un riferimento a un file di metadati o di un errore di chiamata di un file di importazione.  
  
 **ID errore:** BC30141  
  
### Per correggere l'errore  
  
1.  Esaminare il messaggio di errore riportato e vedere l'argomento [Errori e avvisi di Al.exe](http://msdn.microsoft.com/it-it/7f125d49-0a03-47a6-9ba9-d61a679a7d4b) per spiegazioni e suggerimenti aggiuntivi.  
  
2.  Se l'errore persiste, raccogliere informazioni sulla situazione contingente e informare il Servizio Supporto Tecnico Clienti Microsoft.
