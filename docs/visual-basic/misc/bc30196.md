# &#200; prevista la virgola
In un elenco di argomenti manca una virgola \(,\) tra due argomenti.  
  
 **ID errore:** BC30196  
  
### Per correggere l'errore  
  
-   Verificare che tutti gli argomenti siano separati da virgole.
