# Previsto &#39;Compare&#39;
Un'istruzione `Option` specifica `Text` o `Binary`, ma non `Compare`.  
  
 **ID errore:** BC30208  
  
### Per correggere l'errore  
  
-   Aggiungere la parola chiave `Compare` all'istruzione `Option`, ad esempio `Option Compare Binary`.
