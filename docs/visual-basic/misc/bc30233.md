# &#39;&lt;identificatore&gt;&#39; non &#232; valido in una dichiarazione di costante
Un'istruzione `Const` contiene una parola chiave non valida. Un'istruzione `Const` pu√≤ includere solo le parole chiave `Friend`, `Private`, `Protected`, `Public` o `Shadows`.  
  
 **ID errore:** BC30233  
  
### Per correggere l'errore  
  
1.  Verificare che la parola chiave sia stata digitata correttamente.  
  
2.  Rimuovere la parola chiave non valida dall'istruzione `Const`.
