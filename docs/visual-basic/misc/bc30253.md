# Interface&#39; deve terminare con un elemento &#39;End Interface&#39; corrispondente
Un'istruzione `Interface` si verifica senza un'istruzione `End Interface` corrispondente. Ãˆ necessario usare un'istruzione `End Interface` per terminare la definizione dell'interfaccia.  
  
 **ID errore:** BC30253  
  
### Per correggere l'errore  
  
1.  Verificare che la formattazione dei membri `Interface` sia corretta.  
  
2.  Aggiungere un'istruzione `End Interface` alla fine della definizione dell'interfaccia.
