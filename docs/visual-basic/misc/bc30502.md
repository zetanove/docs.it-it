# &#39;Shared&#39; non pu&#242; essere combinato con &#39;&lt;identificatore&gt;&#39; in una dichiarazione di propriet&#224;
Si è provato a combinare il modificatore `Shared` con un identificatore che non è valido in tale combinazione in una dichiarazione di proprietà.  
  
 **ID errore:** BC30502  
  
### Per correggere l'errore  
  
-   Rimuovere l'identificatore.
