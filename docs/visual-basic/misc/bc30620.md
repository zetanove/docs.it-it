# &#39;Option Strict&#39; pu&#242; essere seguito solo da &#39;On&#39; o &#39;Off&#39;
Le istruzioni `Option Strict` devono specificare `On` o `Off`. Tutti gli altri valori non sono validi.  
  
 **ID errore:** BC30620  
  
### Per correggere l'errore  
  
-   Specificare `On` o `Off` per l'istruzione `Option Strict`.
