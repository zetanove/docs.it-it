# L&#39;istruzione &#39;#Region&#39; deve terminare con un &#39;#End Region&#39; corrispondente
Usare la direttiva `#Region` per specificare un blocco di codice da espandere o comprimere durante l'uso della funzionalit√† di struttura dell'editor di codice di [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]. L'inizio e la fine delle istruzioni `#Region` devono essere nello stesso blocco di codice.  
  
 **ID errore:** BC30681  
  
### Per correggere l'errore  
  
1.  Inserire `#End Region` nella posizione appropriata nel codice dopo l'istruzione `#Region`.
