# Prevista fine dell&#39;espressione
Sono stati trovati caratteri aggiuntivi alla fine di un'espressione altrimenti completa.  
  
 **ID errore:** BC30710  
  
### Per correggere l'errore  
  
-   Verificare la sintassi dell'istruzione o dell'oggetto.
