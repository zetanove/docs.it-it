# &#39;Sub New&#39; non pu&#242; implementare membri di interfaccia
`Sub New` è un costruttore e non può implementare membri.  
  
 **ID errore:** BC31042  
  
### Per correggere l'errore  
  
-   Rimuovere le istruzioni `Implements` dalle routine `Sub New`.
