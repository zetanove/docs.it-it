# Per specificare in modo esplicito i valori per tutti gli elementi sono necessarie matrici usate come argomenti di attributo
Una matrice usata come argomento di attributo ha un elemento con un valore non specificato.  
  
 **ID errore:** BC31110  
  
### Per correggere l'errore  
  
-   Specificare tutti i valori dell'elemento.
