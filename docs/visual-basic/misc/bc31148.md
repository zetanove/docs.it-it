# Il prefisso dello spazio dei nomi XML &#39;prefix&#39; non &#232; definito
È stato specificato un prefisso dello spazio dei nomi XML non definito per un valore letterale XML o una proprietà axis XML.  
  
 **ID errore:** BC31148  
  
### Per correggere l'errore  
  
-   Definire il prefisso dello spazio dei nomi XML usando l'istruzione `Imports`.  
  
 \-oppure\-  
  
-   Impostare il prefisso dello spazio dei nomi XML per il valore letterale XML o la proprietà axis XML su un prefisso dello spazio dei nomi XML definito in precedenza.
