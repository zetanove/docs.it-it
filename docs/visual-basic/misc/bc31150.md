# &#200; previsto il tag di fine &lt;/tagName&gt;
Il tag di fine in un valore letterale elemento XML non corrisponde al tag di inizio corrispondente.  
  
 **ID errore:** BC31150  
  
### Per correggere l'errore  
  
-   Modificare il tag XML di fine in modo che corrisponda al tag XML di inizio per il valore letterale elemento XML.
