# L&#39;operatore &#39;If&#39; richiede due o tre operandi
L'operatore `If` accetta solo due o tre argomenti.  
  
 **ID errore:** BC33104  
  
### Per correggere l'errore  
  
-   Esaminare l'espressione `If` per vedere se il numero di operatori pu√≤ essere ridotto o esteso per due o tre argomenti.  
  
-   Si consiglia di usare un'altra costruzione condizionale, ad esempio un'istruzione `If...Then...Else` o un'istruzione annidata `If...Then...Else` per organizzare gli argomenti.
