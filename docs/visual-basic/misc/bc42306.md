# Il tag del commento XML &#39;&lt;tag&gt;&#39; non &#232; consentito in un elemento di linguaggio &#39;&lt;elemento&gt;&#39;
Il tag del commento XML '\<tag\>' non è consentito in un elemento '\<elemento\>'. Il commento XML verrà ignorato.  
  
 Un tag di commento XML non è supportato dall'elemento di linguaggio al quale viene applicato.  
  
 **ID errore:** BC42306  
  
### Per correggere l'errore  
  
-   Rimuovere il tag di commento XML non supportato.
