# Il file esiste gi&#224;
Questo errore si verifica quando a un nuovo file viene attribuito il nome di un file esistente oppure si usa il comando **Salva con nome** per salvare un progetto caricato con un nome già esistente.  
  
### Per correggere l'errore  
  
-   Controllare il nome e sostituirlo con uno che non sia in conflitto oppure eliminare il file o il progetto preesistente se non è più necessario.
