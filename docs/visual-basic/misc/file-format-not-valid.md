# Formato di file non valido
Si è provato a caricare un file in un componente, ma il formato dei dati del file è incompatibile col componente, oppure si è tentato di salvare i dati del componente in un file, ma il formato dei dati è incompatibile con quello del file.  
  
### Per correggere l'errore  
  
1.  Consultare la documentazione del componente per stabilire quale sia il formato corretto per i dati del file su disco.  
  
2.  Consultare la documentazione del componente per stabilire se supporta la conversione da un formato all'altro.
