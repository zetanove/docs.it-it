# La lunghezza dell&#39;argomento &#39;&lt;nomeargomento&gt;&#39; deve essere maggiore di zero
Un argomento ha un valore minore o uguale a zero.  
  
### Per correggere l'errore  
  
-   Specificare un argomento maggiore di zero.
