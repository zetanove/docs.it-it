# L&#39;oggetto non supporta le impostazioni locali correnti (Visual Basic)
Si è provato ad accedere a un oggetto che non supporta le impostazioni locali per il progetto corrente.  
  
### Per correggere l'errore  
  
-   Controllare quali siano le impostazioni locali supportate dall'oggetto.
