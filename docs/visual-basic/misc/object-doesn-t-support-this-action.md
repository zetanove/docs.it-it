# L&#39;oggetto non supporta l&#39;azione (Visual Basic)
Si è fatto riferimento a un metodo o una proprietà che non è supportata da questo oggetto.  
  
### Per correggere l'errore  
  
1.  Per altre informazioni sull'oggetto, vedere la documentazione dell'oggetto.  
  
2.  Verificare che i nomi delle proprietà e dei metodi siano stati digitati correttamente.
