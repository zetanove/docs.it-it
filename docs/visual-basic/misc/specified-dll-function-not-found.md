# Impossibile trovare nella DLL la funzione specificata
La libreria di collegamento dinamico \(DLL\) in un riferimento alla libreria utente è stata trovata, ma la funzione DLL specificata non è stata trovata al suo interno.  
  
### Per correggere l'errore  
  
1.  Specificare un ordinale valido nella dichiarazione di funzione.  
  
2.  Verificare che il nome della DLL e l'alias siano corretti.
