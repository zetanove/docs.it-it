# La chiave del Registro di sistema specificata non esiste
La chiave del Registro di sistema che è stata specificata non esiste.  
  
### Per correggere l'errore  
  
-   Controllare la chiave del Registro di sistema fornita per verificare che il percorso specificato sia corretto.  
  
-   Creare la chiave del Registro di sistema prima di tentare di scrivervi.
